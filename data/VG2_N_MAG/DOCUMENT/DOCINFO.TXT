The /DOCUMENT directory contains two subdirectories each of which contains    
a document relevant to the Voyager mission and its instruments. An ASCII text 
version with all graphics removed (file extension .ASC) is provided for each  
document, along with a PDF version.                                           
                                                                              
/DOCUMENT directory tree structure:                                           
                                                                              
  [DOCUMENT]                                                                  
     |                                                                        
     |- DOCINFO.TXT   (this file) describes contents of the /DOCUMENT         
     |                directory tree                                          
     |                                                                        
     |                                                                        
     |- [INSTRUMENT_PAPER]  contains copies of the Voyager MAG investigation  
     |                description.                                            
     |                                                                        
     |                                                                        
     |- [MISSION_DESCRIPTION] contains copies of the Voyager mission          
     |                description paper.
